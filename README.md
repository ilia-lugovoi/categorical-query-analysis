# Тематический анализ самых популярных запросов в Яндекс Картинках

Тестовый проект по анализу поиска в Яндекс Картинках

## Цель проекта

Провести EDA, ответив на вопросы заказчика, подготовить датасеты и создать ценный дашборд в Datalens

**Вопросы заказчика:**

1. Определить диапазон дат в предоставленном датасете
2. Посчитать количество запросов `"ютуб"` по платформам `desktop` и `touch`
3. Выделить топ-10 самых частотных запросов по платформам и сравнить их
4. Исследовать, как отличается трафик запросов в течение дня
5. Выделить контрастные тематики запросов для `desktop` и `touch`

**Итоговоый дашборд:**

https://datalens.ru/3dcmiy3thxban

## Структура проекта

### Основной ноутбук

Главный файл проекта: [yandex_images_analysis.ipynb](./yandex_images_analysis.ipynb)

Ноутбук устроен как последовательный аналитический пайплайн:

1. Загрузка и первичная подготовка данных
   - чтение `data.tsv`
   - приведение временной метки к дате и часу
   - создание технических полей для анализа

2. Ответы на вопросы тестового
   - диапазон дат

3. Тематический анализ
   - выделение частотных запросов
   - ручная категоризация запросов
   - расчет контрастов тематик между `desktop` и `touch`

4. Подготовка данных для дашборда
   - витрина по времени
   - grouped-витрина по темам
   - витрины для почасового и недельного паттернов

## Логика тематического анализа

Для тематического анализа я сознательно использовал не весь хвост распределения, а наиболее частотную часть запросов.

В проекте:

- выделены запросы с частотой `300+` за весь период
- вручную проведена тематизация этих запросов
- на этой выборке построен контрастный анализ между `desktop` и `touch`

Такой подход не покрывает весь трафик, но хорошо подходит для анализа наиболее заметной и продуктово значимой части пользовательского спроса.

## Ключевой вывод

Гипотеза подтверждается: интересы пользователей поиска по картинкам на `desktop` и `touch` заметно различаются.

- `desktop` сильнее связан с учебными, рабочими и справочными сценариями
- `touch` сильнее связан с поздравлениями, развлечениями и быстрыми повседневными задачами
